# Финальный проект 1 семестра

REST API сервис для загрузки и выгрузки данных о ценах.

## Требования к системе

### Необходимое программное обеспечение

- **Операционная система:** Совместимый с Go и PostgreSQL. Рекомендуется использовать Linux, macOS или Windows.
- **Go** (версия 1.16 или выше): [Установить Go](https://golang.org/doc/install)
- **PostgreSQL** (версия 12 или выше): [Установить PostgreSQL](https://www.postgresql.org/download/)

### Аппаратные требования

- CPU: 1 ядро или выше
- RAM: 512 MB или выше
- Место на диске: 250 MB свободного пространства

## Установка и запуск

1. **Клонируйте репозиторий:**

   
   git clone https://github.com/ваш_репозиторий/your-repo.git
   cd your-repo
   
2. **Настройте базу данных PostgreSQL:**

   Создайте базу данных и таблицу, используя команды ниже через psql:

   
   CREATE DATABASE project_sem_1;
   \c project_sem_1
   CREATE TABLE prices (
       id SERIAL PRIMARY KEY,
       product_id INT,
       name VARCHAR(255),
       category VARCHAR(255),
       price NUMERIC,
       create_date DATE
   );
   
3. **Запустите приложение:**

   Убедитесь, что у вас установлен Go. В корневом каталоге проекта выполните команду:

   
   go run main.go
   
   Сервер будет запущен по адресу localhost:8080.
